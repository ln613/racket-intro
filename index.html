Program

    • Wait for data input (from keyboard/mouse/file/db/network/camera/mic/sensor...)

Porgramming Language

  • Functional Specification (Business Analysis)
    • Human readable, plain english
    • click addToCart => add current product to cart
  • Programmer: translate FS to a program using a PL
  • The compiler of the PL compiles the program to CPU instructions

Program Structure

  • Each line is one of:
    • Definition (define, define-struct)
    • Expression
    • Test case (check-*)
    • Library (require)
  • BSL Doc

Definition

  • Function definition
    • (define (name v1 v2 ...) body-expr)
  • Constant/Variable definition
    • (define name expr)
  • Structure definition
    • (define-struct name (field-name ...))

Expression

  • Value
    • number/string/symbol/boolean/character/list
    • constant/variable
  • Function calls
    • (name expr1 expr2 ...)
    • same number of parameters as definition
    • Primitive functions (+/-/*/string-append/list/car...)
  • Evaluate to a value

Order of Evaluation

  • Parameter first, left to right
  • if/cond/and/or

Number

  • exact
    • integer, rational (2/3), integer complex
  • inexact
    • float (1.0), float complex
  • primitive operations
    • +, -, *, /, abs, max ..., =, >, < ...
    • number functions
    • number?, integer?, exact?, zero?, inexact->exact ...
    • number type functions

Boolean

  • #t, #f, #true, #false, true, false
  • anything other than #f is true (if 0 ...)
  • primitive operations
    • boolean?, not, and, or ...
    • boolean functions

Equality

  • eq? is pointer comparison (twins are not equal)
  • equal? structually equal (twins are equal)
    • (equal? '(1 a) '(1 a)) -> #t
    • "=" is the same as equal? but only works on numbers
  • eqv?
    • the same as eq? on string, list
    • the same as equal? on number, character
    • (eqv? (/ 2 3) 2/3) -> #t
  • (eq? '() '()) -> #t

Character

  • #\a
  • char->integer, integer->char
  • Compare: char=?, char<?, char>? ...
  • Category: char-numeric?, char-lower-case? ...
  • Convert: char-upcase, char-downcase ...
  • (eq? #\λ (string-ref "λ" 0))
  • character functions

String

  • An array of characters, 0 based
  • mutable or immutable
  • Symbol = String, but no twins
  • Basic: string-length, string-ref, string-append, substring, string->list ...
  • Compare: string=?, string<?, string>? ...
  • Convert: string-upcase, string-titlecase ...
  • Other: string-join, string-split, string-replace, string-trim, string-contains? ...
  • string functions

Pair & List

  • '(1 "a1" a1 #t (a b))
  • Pair: a list with 2 values
  • '() = empty = null
  • Basic: list?, null?, pair?, length, cons, append, list ...
  • Select: list-ref, car/first, cdr/rest, caar, cadr, caaaar ...
  • Lambda: map, andmap, filter, foldl, foldr, sort ...
  • list functions

Struct

  • (define-struct point (x y))
  • (define p1 (make-point 3 4))
  • (point-y p1)
  • struct functions
